# AgriBuyX – Project Overview

**AgriBuyX** is a modern agricultural marketplace platform built with Next.js and deployed on Vercel.  
It is designed to operate as a comprehensive platform connecting farmers, vendors, and buyers.

Currently in **Phase 1 (Lite)**: Only internal admins/developers can upload products.  
Public browsing is allowed, but external vendors cannot sign up unless invited.

AgriBuyX sets the foundation for the full agricultural commerce ecosystem.

---

## 1.1 Purpose of AgriBuyX
- Launch quickly with a functional agricultural marketplace
- Develop modern infrastructure with Next.js + Vercel
- Gather early traffic, data, and user engagement
- Prepare for vendor onboarding and advanced features
- Build brand awareness with SEO-optimized platform

---

## 1.2 Key Characteristics
- Lightweight  
- Fast to develop  
- Zero/low cost hosting  
- Easy to grow into the full AgriBuy system  
- Controlled uploads (admins only)  
- Secure and stable foundation  

---

## 1.3 Scope of AgriBuy Lite (Phase 1)

### Core Features
- Admin dashboard for product uploads  
- Product listing pages  
- Product detail pages  
- Category filtering  
- Location tagging  
- Image watermarking  
- WhatsApp/Telegram contact buttons  
- Invite-only vendor registration (optional)

### Users
- **Admins/Developers** – full management access  
- **Public Buyers** – browse only  
- **Invite-only Vendors** – future small phase  

---

## 1.4 AgriBuy Lite Goals
- Launch a working MVP with very minimal cost  
- Deploy frontend + backend on Vercel  
- Integrate with Supabase for database, auth, and storage  
- Build initial audience and traffic  
- Validate design and workflow for the full AgriBuy platform

---

## 1.5 Out of Scope (For Lite Version)
- Public vendor sign-up  
- Payments or escrow  
- Logistics/delivery  
- Vendor storefronts  
- Subscription/boosted ads  
- Full mobile app  
- Review & rating system  

These will be introduced in **AgriBuy Core** and **AgriBuy Pro**.

---

## 1.6 AgriBuy Roadmap
### Phase 1 — AgriBuy Lite (Current)
Internal-only uploads, lightweight product browsing, minimal backend structure.

### Phase 2 — AgriBuy Core
Full vendor onboarding, dashboard, buyer accounts, mobile app, messaging, etc.

### Phase 3 — AgriBuy Pro
Payments, escrow, delivery partners, farm insights dashboard, premium ads, etc.

---

AgriBuy Lite is the foundation upon which the complete AgriBuy ecosystem will be built.
