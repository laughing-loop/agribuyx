# Google Gemini Code Assist -- VS Code Integration Guide

## 1. Create a Google Account

You will need a Google account to use Gemini Code Assist.

## 2. Enable Gemini for Developers

Visit: https://ai.google.dev\
Enable the API and create a project in Google Cloud Console.

## 3. Create an API Key

1.  Go to Google Cloud Console.
2.  Navigate to **APIs & Services \> Credentials**.
3.  Click **Create Credentials \> API Key**.
4.  Save your API key.

## 4. Install the Gemini Extension in VS Code

1.  Open **Visual Studio Code**.
2.  Go to **Extensions**.
3.  Search for **"Gemini Code Assist"**.
4.  Click **Install**.

## 5. Configure the Extension

After installing: 1. Press **Ctrl + Shift + P**. 2. Search **"Gemini:
Set API Key"**. 3. Paste your API key.

## 6. Start Using Gemini

-   Use **inline code completions**.
-   Ask Gemini to generate full pages of code.
-   Use it for TypeScript, Next.js, NestJS, and more.
-   Free tier includes generous usage for personal projects.

## 7. Free Tier Details (as of 2025)

-   Free monthly quota for code completions.
-   May vary depending on region/project.
-   Ideal for learning, building MVPs, and experimenting.

## 8. Notes

-   If limits are exceeded, upgrade to a paid plan.
-   Free tier is enough for student, hobby, and small project workloads.
